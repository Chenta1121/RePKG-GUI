关于 RePKG 批量提取 GUI 使用说明与特性

===================================

核心功能特性:
- 批量处理: 自动扫描指定根目录下所有 .pkg 文件，并依次调用 RePKG 进行提取或信息查询。
- 路径配置: 可配置 RePKG.exe 路径、输入根目录和输出根目录，并自动保存配置。
- 命令选项: 通过 GUI 界面灵活勾选 RePKG 的命令行参数，如转换 TEX、复制项目文件等。
- 命令预览: 实时生成 Windows CMD 格式的完整命令，方便用户确认。
- 日志系统: 详细记录每次任务的执行过程、RePKG 的输出以及任何错误信息。

Python 脚本增强功能:
- 预览图像拷贝: 在提取完成后，自动查找并拷贝同名的 preview.* 或 .pkg 文件同名的图像文件到输出目录。
- 自动分类: 根据提取出的 project.json 文件中的 type 字段，自动将项目移动到对应的分类子文件夹中（如 scene, video, Unknown）。
- 透明映射: 在分类完成后，可创建符号链接，将分类后的项目映射回输出根目录，实现“分类目录”和“未分类外观”共存的效果。

使用步骤:
1. 在“配置设置”页，设置 RePKG.exe 路径、输入根目录和输出根目录。
2. 根据需求勾选“命令选项”和“Python 脚本选项”。
3. 点击 “运行任务”，程序将在“运行日志”标签页开始批量处理。
4. 任务完成后，您可以使用 “分类项目” 和 “建立映射” 功能来整理您的输出目录。

===================================

⚠️ 注意事项与故障排除:

关于符号链接：
在 Windows 系统中，创建符号链接（即“透明映射”）通常需要管理员权限运行本程序，或在系统设置中启用开发者模式。如果创建映射失败，请检查您的运行权限。

乱码问题：
如果 RePKG 的输出日志中出现乱码，请确保您的 RePKG 版本支持 UTF-8 编码，并且您的 Windows CMD/PowerShell 编码设置正确。

程序更新：
请关注本工具的最新版本发布，以获取最新的功能和错误修复。
